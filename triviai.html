TRIVIAI

TriviAI is a trivia app powered by AI. This project provided me with valuable experience working with OpenAI’s API, React and TailwindCSS.

Live Site GitHub Repo

Features

  • Dynamic Question Generation: Leverages OpenAI’s API to create unique trivia questions for any topic you choose.
  • Customizable Features: Enter any topic, choose your difficulty level and number of questions then watch the app generate engaging trivia tailored to your choices.

Technology

  • Front End: React, Vite
  • Styling: TailwindCSS
  • API: OpenAI API
  • Deployment: GitHub Pages and Heroku Proxy Server
